Transforming Code

Much of programming is simply mapping one kind of code to another. For example, you have to get your forms lined up with your state management system and your networking code to be in sync with your API.

Optic automates these mappings with our Transformations which appear in the Search Panel whenever they are available for the code you are interacting with. Transformations take one code object and return another of a known type.
